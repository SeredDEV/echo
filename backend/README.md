# Backend - Echo Tecnología

Esta carpeta contendrá el backend del proyecto Echo Tecnología.

## Estructura Futura

```
backend/
├── src/
│   ├── controllers/
│   ├── models/
│   ├── routes/
│   ├── middleware/
│   └── config/
├── package.json
└── README.md
```

## Tecnologías Sugeridas

- **Node.js** + **Express** (API REST)
- **MongoDB** o **PostgreSQL** (Base de datos)
- **JWT** (Autenticación)

## Próximos Pasos

1. Inicializar proyecto Node.js: `npm init`
2. Instalar dependencias necesarias
3. Configurar estructura de carpetas
4. Implementar endpoints de API
5. Conectar con base de datos
